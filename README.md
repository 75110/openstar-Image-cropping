# 批量图片分割工具 (Rust)

使用 Rust + egui 开发的高性能批量图片分割工具。体积更小、启动更快、内存占用更低。

## ✨ 特性

- 🚀 **极速启动**：原生 Rust 性能，秒开体验
- 🎨 **现代 UI**：基于卡片式布局的现代设计
- 📦 **轻量体积**：编译后仅约 5-10MB
- ⌨️ **高效操作**：支持快捷键、多选分割线、键盘微调
- ⚡ **并行处理**：多线程批量导出，效率倍增

## 🛠️ 构建与运行

### 环境要求
- Rust 1.75+

### 快速开始
```bash
# 运行程序
cargo run --release
```

## ⌨️ 操作快捷键

| 操作 | 快捷键 |
|------|------|
| 选择文件 | `Ctrl + O` |
| 切换图片 | `Ctrl + ← / →` |
| 保存配置 | `Ctrl + S` |
| 批量处理 | `Ctrl + Enter` |
| 微调分割线 | `↑ / ↓ / ← / →` (需先选中线) |
| 多选分割线 | `Ctrl + 点击` 或 `鼠标框选` |
